# Sales Data Analysis Project

## Project Overview

This project analyzes sales data to identify revenue trends, profitability,
customer behavior, product performance, and market insights using:

- SQL
- Excel
- Power BI

## Tools Used

- SQL (Data Analysis)
- Microsoft Excel (Exploratory Analysis)
- Power BI (Interactive Dashboard)

## Dataset

The dataset contains:

- Order details
- Customer information
- Product details
- Sales
- Profit
- Discount
- Shipping information

## Project Files

### Dataset
Cleaned dataset used for analysis.

### SQL Analysis
Contains queries for:

- Sales analysis
- Market analysis
- Product performance
- Customer analysis

### Excel Analysis

Includes:

- Executive summary
- Category analysis
- Region analysis
- Market analysis
- Customer analysis

### Power BI Dashboard

Interactive dashboard containing:

- Total Sales
- Total Profit
- Total Orders
- Average Discount
- Sales Trend
- Category Performance
- Market Analysis
- Top Products
- Customer Analysis

### Business Insights Report

PDF report summarizing key findings and recommendations.

## Key Insights

- Identified top-performing regions and markets
- Analyzed profitable categories and sub-categories
- Found high-value customers
- Studied monthly sales trends
- Evaluated discount impact on profitability

## Recommendations

- Focus on high-performing markets
- Optimize discount strategies
- Improve low-profit product categories
- Target valuable customer segments
